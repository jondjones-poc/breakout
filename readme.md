## 👾 How To Use ☄️

Terminal into your new projects root folder:

```bash
npm install
```

Start development server:

```
npm run start
```

This will run the game locally on port 8000.  To create a production build:

```
npm run build
```

Production files will be placed in the `dist` folder.
